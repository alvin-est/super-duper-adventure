## Installation
1. Install PostgreSQL and NodeJS.
2. Clone the repo.
3. Navigate to the folder.
4. Run the commands:
      - npm i
      - npm run reset
      - npm run seed

5. Start the server.

## Usage
To use this app, you must first start the server:
1. Run the command:
      - npm run start 
2. Navigate to the web URL 
3. You will need to create an account to start posting.
